# Belgian Company Search (Laravel)

A Laravel web application inspired by Companyweb that allows users to search Belgian companies by **VAT number or company name**.

The application uses **real Belgian company data from BCE/KBO Open Data** and allows users to validate VAT numbers using the **European VIES service**.

---

# Features

- Search Belgian companies by **VAT number**
- Search Belgian companies by **company name**
- **Autocomplete suggestions** while typing
- **Pagination** of search results
- **Company detail page**
- **Copy VAT number button**
- **VAT number validation via VIES**
- Import **real Belgian companies from BCE Open Data**
