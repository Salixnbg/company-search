# Belgian Company Search (Laravel)

This project is a simple Laravel application inspired by Companyweb.

The application allows users to search for Belgian companies using either the VAT number or the company name.

## Features

- Search companies by VAT number or company name
- VAT number normalization (different formats supported)
- Autocomplete suggestions while typing
- Pagination of search results
- Detailed company information page
- Copy VAT number button
- Import companies from CSV file

## Technologies

- Laravel
- PHP
- MySQL
- Blade
- Bootstrap
- JavaScript (AJAX)
- Git
