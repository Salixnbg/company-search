# Belgian Company Search

A Laravel web application that allows users to search for Belgian companies by **company name**, **enterprise number**, or **VAT number**.

The application also includes an **admin dashboard** that allows managing companies through a complete **CRUD interface**.

---

# Live Demo

The application is available online at:

**https://belgian-company-search.onrender.com**

---

# Features

### Public features

- Search Belgian companies by **name**
- Search by **enterprise number**
- Autocomplete suggestions while typing
- Company detail page
- Display of:
  - company name
  - VAT number
  - enterprise number
  - legal form
  - address
  - city
  - postal code
  - company status
- Copy VAT number button
- Responsive mobile design

---

### Admin features

Admin dashboard with full CRUD:

- Create company
- Edit company
- Delete company
- Search companies
- Alphabetical sorting
- Pagination (50 companies per page)

The admin interface is **protected by authentication**.
